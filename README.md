# Marquee App - 開源 Android 跑馬燈

一個基於 Android 平台、使用 Kotlin 與 Jetpack Compose 開發的現代化跑馬燈應用程式。
具備高度客製化的文字動態效果與直覺的 UI 操作介面。

## 📱 核心功能特色

本應用程式提供全方位的跑馬燈自訂選項，讓使用者能針對不同場景（如演唱會應援、活動公告）打造專屬的動態文字：

* **📐 字體調整**：可即時調整大小、粗細、捲動方向，確保在各尺寸螢幕上皆具備最佳閱讀性。
* **⚡ 速度動態調整**：可變更文字滾動速度，從流暢慢速到快速皆可控制。
* **🎨 色彩自訂**：提供獨立調色盤與預設色塊，支援動態變更字體色彩以及文字閃爍功能。
* **🔄 動態即時預覽**：上方預覽區域與下方控制面板連動，任何屬性修改（文字、顏色、速度）皆能即時反映「所見即所得」。
*  **🕹️儲存功能**：將目前調配好的跑馬燈參數組合（包含文字內容、滾動速度等）一鍵儲存，建立個人化清單。

## 🚀 開發方法與實作
**AI 輔助開發** 模式進行開發。由開發者負責核心功能設計、UI 互動邏輯規劃與系統架構定義，並善用 AI 工具進行程式碼建構、重構與 Bug 排除。
* **提示詞工程應用**：將 Compose 狀態管理拆解為具體邏輯需求，透過高準度提示詞引導 AI 生成 UI 程式碼。
* **快速建構**：在短時間內透過 AI 進行技術驗證，將核心精力聚焦於使用者體驗與功能完整性。
* **除錯與重構**：主動審查 AI 生成之程式碼，確保符合專案架構。
