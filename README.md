# Marquee App - 智慧型手機跑馬燈

一個基於 Android 平台、使用 **Kotlin** 與 **Jetpack Compose** 開發的現代化跑馬燈應用程式。本專案具備高度客製化的文字動態效果與直覺的 UI 操作介面，並完美導入了現代 Android 開發的主流架構與 Git 版本控制規範。

## 📱 核心功能特色

本應用程式提供全方位的跑馬燈自訂選項，讓使用者能針對不同場景（如演唱會應援、活動公告、機場接機）打造專屬的動態文字：

* **⚡ 速度動態調整**：支援透過 Slider 變更文字滾動速度，從流暢慢速到強烈快速皆可精準控制。
* **🎨 豐富色彩自訂**：
    * **文字顏色**：提供獨立調色盤與預設色塊，支援動態變更字體色彩。
    * **背景色彩**：可自由切換純黑、純白或各式主題色，襯托文字視覺。
* **📐 字體與佈局優化**：內建字體大小（Size）即時縮放調整，確保在各尺寸螢幕上皆具備最佳閱讀性。
* **🔄 動態即時預覽**：上方預覽區域與下方控制面板連動，任何屬性修改（文字、顏色、速度）皆能即時反映，達到「所見即所得」。
* **🧹 一鍵重置機制**：內建「重置」按鈕，可快速將所有參數還原至系統預設值。

## 🛠️ 技術棧與架構

* **開發語言**：Kotlin
* **UI 框架**：Jetpack Compose (Declarative UI 宣告式架構)
* **狀態管理**：Compose State / MutableState 機制（確保 UI 與資料狀態流暢同步）
* **版本控制**：Git / GitHub
